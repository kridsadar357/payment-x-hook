# pwa-dashboard

Next.js 14 (App Router) + TypeScript + Tailwind · `src/app` layout — **full scaffold on disk** (not an empty folder).

## Commands

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## PWA

`public/manifest.json` is included. Add PNG icons under `public/` when you want polished install prompts.
